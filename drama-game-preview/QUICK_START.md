# 🚀 剧情织造机 - 快速上手指南

## 📋 5分钟快速体验

### 步骤1：添加第一个场景
1. 点击底部导航的 **📍 场景**
2. 点击右上角 **➕ 新增**
3. 填写信息：
   - 名称：`神秘森林`
   - 描述：`月光透过树叶洒下，周围传来奇怪的声音`
   - 标签：`森林, 夜晚, 神秘`
4. 点击 **保存**

### 步骤2：创建故事层级
1. 点击底部导航的 **📚 层级**
2. 确认已有三个默认层级（开场准备 → 核心解谜 → 结局判定）

### 步骤3：添加玩法
1. 点击底部导航的 **📖 玩法**
2. 点击 **➕ 新增**
3. 填写信息：
   - 名称：`寻找光源`
   - 描述：`在黑暗中摸索，试图找到光源`
   - 结果：`发现了一盏油灯`
   - 所属层级：`开场准备`
   - 标签：`探索, 生存`
4. 点击 **保存**

### 步骤4：体验故事生成
1. 点击底部导航的 **▶️ 生成**
2. 选择 **神秘森林** 场景
3. 点击 **一键生成流程** 或 **开启剧情走向**
4. 享受您的第一个故事！

## 🎯 模板推荐

### 模板1：恐怖密室逃脱

**场景**：
- 密室：`昏暗的密室，只有一扇锁着的门`（标签：`室内, 恐怖, 密闭`）
- 走廊：`长长的走廊，墙上有奇怪的涂鸦`（标签：`室内, 恐怖, 狭长`）

**玩法**：
- 搜寻钥匙：`仔细寻找可能隐藏的物品`
- 破解密码：`墙上有一串数字需要破译`
- 开启机关：`发现了一个隐藏的开关`

**指令**：
- 突发噪音：`门外传来诡异的脚步声`（概率：30%）
- 灯光闪烁：`灯光突然开始闪烁不定`（概率：40%）

### 模板2：奇幻冒险

**场景**：
- 魔法森林：`充满魔法能量的森林，植物会发光`（标签：`森林, 奇幻, 魔法`）
- 古老遗迹：`被遗忘的神殿遗迹，充满神秘力量`（标签：`遗迹, 奇幻, 古老`）

**玩法**：
- 学习魔法：`向森林中的精灵学习基础魔法`
- 寻找宝物：`在遗迹中寻找传说中的宝物`
- 解除封印：`发现了一个被封印的古老魔法`

**指令**：
- 魔法暴动：`魔法能量突然失控`（概率：20%）
- 精灵出现：`神秘的精灵突然出现`（概率：35%）

## 💡 创作灵感

### 故事主题构思

**侦探故事**：
- 场景：犯罪现场、警察局、证人住所
- 玩法：搜查证据、询问证人、分析线索
- 指令：神秘电话、新发现、突发线索

**校园生活**：
- 场景：教室、图书馆、操场、食堂
- 玩法：上课听讲、社团活动、朋友聚会
- 指令：突发考试、偶遇偶像、意外事件

**科幻冒险**：
- 场景：太空站、外星基地、实验室、飞船
- 玩法：驾驶飞船、研究科技、外星交流
- 指令：系统故障、外星攻击、科学发现

### 剧情转折技巧

1. **意外发现**：通过指令触发意外事件
2. **身份揭示**：在关键层级揭露重要信息
3. **道德选择**：设计没有正确答案的玩法选择
4. **时间限制**：添加紧迫感的突发指令

## 🎮 测试您的作品

### 快速测试流程

1. **单元素测试**：
   - 只添加一个场景和几个玩法
   - 体验基础生成效果
   - 调整标签和概率

2. **完整故事测试**：
   - 创建完整的3-5层结构
   - 测试快速模式和沉浸模式
   - 检查故事连贯性

3. **用户体验测试**：
   - 从玩家角度体验故事
   - 检查难度平衡
   - 确认指令触发合理

### 优化建议

- **标签优化**：确保场景和玩法标签匹配度高
- **概率调整**：根据测试结果调整指令触发概率
- **流程优化**：检查每层玩法的丰富程度
- **文本润色**：优化描述文字的表现力
